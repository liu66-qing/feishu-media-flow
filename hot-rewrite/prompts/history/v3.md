# hot-rewrite prompt v3

## 目标

优化 Simhash 相似度计算方式，并保留原始分数。

## 调整

- 使用 Simhash 计算文本相似度
- 输出 similarity_score 作为归一化后分数
- 同时输出 simhash_raw_score 保留原始分数
- 输出 similarity_method 说明计算方式

## 原因

测试发现，Simhash 原始公式对中文同主题短文本偏高，raw_score 在 0.5 附近并不一定代表高度相似。因此使用 0.5 作为随机基线，计算 normalized similarity_score。

## 当前结果

case_001 最终通过：

- similarity_score < 0.3
- source_attribution 非空
- rewritten_content 与原文角度不同
