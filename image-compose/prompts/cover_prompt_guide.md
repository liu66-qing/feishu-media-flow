# 小红书封面背景AI生成Prompt指南
## 说明
本指南用于AI生成小红书封面背景图，**所有背景图严格禁止出现任何文字、字母、数字、水印、logo、清晰可识别的人脸**，文字内容统一通过`image-compose`模块的HTML+CSS+Pillow模板叠加合成。
所有生成图统一使用**3:4竖版比例（1242×1656px）**，完美适配小红书封面尺寸。

### 中文通用模板（适配豆包/通义万相等国内大模型）


## 三、排版/构图/留白关键词
### 留白位置（必选，决定文字叠加位置，适配现有模板）
- **中心留白**：画面中心是大面积干净低对比度区域，装饰元素分布在画面四角，适配现有`xhs-cover-01/02/03/04`等标题居中的模板，最常用
- **上半部分留白**：场景元素集中在画面下半部分，上半部分干净明亮，适配标题在顶部的模板
- **下半部分留白**：场景元素集中在画面上半部分（如天空、树叶），下半部分干净，适配标题在底部的模板
- **侧边留白**：场景元素集中在画面左侧/右侧，另一侧干净，适配标题左对齐/右对齐的模板
- **全幅柔焦留白**：整个画面是大光圈虚化的场景，整体对比度低，文字叠加在任意位置都清晰，适配复杂排版模板

### 构图关键词
- 居中对称构图：元素对称分布在四角，中心留白，视觉稳定正式
- 对角线构图：元素沿对角线分布，留白在另一侧，活泼有动感，适合日常/活动类
- 框架构图：用树叶、窗户、书架等元素形成自然边框，中间留白，氛围感强
- 俯拍平铺构图：元素平铺在桌面四周，中心留白，适合好物/美食/穿搭类
- 浅景深构图：背景/前景大光圈虚化，焦点外区域都是柔和的“天然留白”，文字叠加不冲突

### 光影关键词
- 柔和漫反射光：窗边自然光，阴影柔和不生硬，是最通用不出错的光影
- 丁达尔光：阳光透过树叶/窗户形成光束，氛围感强，注意光束不要遮挡留白区域
- 暖黄台灯光：夜晚/宿舍场景，温馨治愈，阴影柔和
- 冷白柔光：极简/数码/正式类内容，干净利落，无多余阴影
- 逆光剪影：风景/情绪类内容，亮部作为留白区域，氛围感强
- 雾面柔光：莫兰迪/高级感内容，无硬阴影，整体质感柔和

### 画质&避坑关键词
- 正向：8K高清、细节细腻、锐化适中、色彩自然、氛围统一、对比度适中、无过曝、无暗部死黑
- 反向禁止：任何文字/字母/数字、水印logo、清晰人脸、杂乱多余元素、高饱和度艳俗颜色、高对比度硬阴影抢文字视线

## 四、参考封面风格分类
常用封面分为5类核心风格，对应关键词可以直接复用：
1. **纯色简约风**：对应现有基础模板风格，AI生成时使用「莫兰迪渐变、哑光质感、极简几何小装饰、中心大面积留白」
2. **学习场景风**：书桌、笔记、文具类封面，使用「ins书桌、柔和自然光、浅景深虚化、中心留白」
3. **清新校园风**：校园风景、树叶光影类封面，使用「梧桐道光斑、丁达尔光、背景虚化、上半部分留白」
4. **可爱手账风**：马卡龙色、便签装饰类封面，使用「手账风、奶油色、卡通边角装饰、中心留白」
5. **撞色活力风**：高饱和活泼类封面，使用「多巴胺撞色、几何小元素、活泼动感、中心留白」
