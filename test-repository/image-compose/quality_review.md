## 检查项

- [ ] 输出 PNG > 0 字节
- [ ] 尺寸像素正确（1080x1350）
- [ ] 10字/20字/30字标题不截断
- [ ] 中文字体无方块/乱码
- [ ] 4 套模板都能独立调用
- [ ] 错模板名返回 error.json
- [ ] 文字对齐一致
- [ ] 颜色搭配不刺眼
- [ ] 留白合理（文字区不超过画面 60%）
- [ ] 字体层级分明（标题 > 副标题）
- [ ] 不像 Word 艺术字
