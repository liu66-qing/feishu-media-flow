## 标题（复制）
踩了几万的雷才明白，大学创业该怎么做

## 正文（复制）
去年我跟几个同学盘下一家奶茶店，结果三个月亏光了本金。

那段时间我天天失眠，看着账本上的负数真的很难受。

作为过来人，今天想掏心窝子跟你们聊聊创业这件事。

先说说项目选择，千万别盲目跟风开奶茶店或者快递驿站。

这种赛道早就卷成红海，新手入场成功率真的极低。

我亲眼见过同一条街半年倒了四家新店，别拿生活费去试水。

另外一定要先做小成本试错，别一上来就砸几万块装修进货。

你可以先摆个周末市集，或者在朋友圈跑通一次完整的交易闭环。

亲测用几百块验证需求，比闷头投钱稳妥得多。

还有一点很重要，找合伙人一定要能力互补且价值观一致。

我当初只图关系好就拉室友合伙，结果分工扯皮直接散伙。

真正靠谱的搭档，是能在遇到分歧时坐下来聊透规则，而不是互相甩锅。

对了，千万别忽略学校自带的创业孵化基地和各类补贴。

很多新生其实都不知道，申请免费工位能省下一大笔启动资金。

去团委或者创新创业学院跑一趟，你会发现比自己在外面租房划算得多。

最后想特别提醒的是，创业再忙也别挂科，毕业证真的很重要。

我身边有为了搞项目连旷两周专业课的，最后拿延期毕业换教训。

学业才是你的基本盘，项目黄了可以重来，学历丢了可补不回来。

回头看这些跟头，其实都是青春必经的弯路。

跌倒了不可怕，关键是学会怎么爬得更稳。

希望我的教训能帮你省下真金白银，稳稳地往前走。

你现在有具体的创业想法吗，来评论区聊聊，我帮你一起捋捋。
