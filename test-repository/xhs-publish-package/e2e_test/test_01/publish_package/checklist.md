招新没人来？学长掏心窝子给4条建议

## 正文（复制）
去年九月社团招新，我负责的摊位摆了整整一下午。

发出去的五百张黑白传单，倒下一大半全进了旁边的垃圾桶。

看着隔壁被人群围满的招新点，我当时急得直搓手。

那天秋老虎特别晒，摊位前原本热闹的人群没撑半小时就散了个干净。

后来跟带我的学姐复盘才搞明白，光靠发单页根本留不住人。

先说说预热这个动作，千万别等出摊了才临时吆喝。

提前三天在校园表白墙、新生群或者小红书发预告，效果截然不同。

我们试着提前透了核心活动的排练花絮，当天来咨询的人数直接翻了一倍。

另外就是宣传物料的质量，这步真的不能太抠预算。

我亲测对比过，设计用心的精美折页，被带回寝室的概率比黑白单页高了五倍。

排版留白多一点，把招新群二维码和核心福利放在最显眼的位置，学弟学妹扫码的意愿会强很多。

到了摆摊现场，最忌讳的就是干巴巴地递纸。

一定要在摊位旁边腾出一块体验区，放点你们社团的硬核装备。

只要新生愿意停下来亲手试一下，他们对社团的好奇心立马就勾起来了。

除此之外，现场安排点小互动真的能快速破冰。

搞个简单的知识问答或者投壶游戏，填完意向表就送定制小周边。

我们去年靠几块钱的帆布袋，硬是拉来了四十多个高质量新成员。

最后想特别提醒的是，摊位前的气场要靠人来撑。

让几个靠谱的老成员统一换上社团文化衫，站在最前面直接演示常规活动。

视觉冲击力一旦拉满，路过的人根本挪不开眼。

招新从来不是拼运气，而是拼谁更懂怎么展示诚意。

把细节做到位，对的人自然会被你们吸引过来。

看着新一批学弟学妹在群里慢慢活跃起来，那种成就感真的无可替代。

今年你们招新准备了什么特别环节呀？来评论区跟我聊聊吧。
