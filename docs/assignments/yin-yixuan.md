# 尹羿璇：平台偏好分析与生成风格适配

本文所称三个平台固定指：公众号、抖音、小红书。

## 负责目标

根据真实平台样本，总结公众号、抖音和小红书在选题、语言和画面方面的偏好，并让同一个选题在不同平台呈现出真正不同的表达方式。

这部分的重点不是继续增加固定模板，而是让平台近期的真实内容表现能够影响后续生成结果。

## 当前已有基础

- 公众号、抖音和小红书已有各自的内容生成模块。
- 已有平台语言、内容长度、标题和内容结构的静态规则。
- 已有内容质量与平台适配评审。
- 已有校园编辑海报、校园手绘和AI校园背景等视觉能力。
- 已有根据选题语义选择视觉风格的能力。

当前规则主要来自人工预设，还没有依据近期平台样本自动形成偏好画像。画面风格也主要跟随选题，而不是同时考虑目标平台。

## 具体任务

- 分析公众号、抖音和小红书近期高表现内容的共同特征。
- 总结各平台更偏好的选题类型、标题结构和内容切入角度。
- 总结各平台的语言语气、段落节奏、内容长度和互动方式。
- 分析封面构图、主色、人物比例、留白、视觉焦点和图片数量。
- 建立三个平台相互独立的偏好画像。
- 保证每条偏好结论都有真实样本和来源依据。
- 将平台偏好应用到现有文案、画面和内容评审中。
- 保留已经完成的校园视觉体系，不因新增偏好分析破坏现有能力。
- 避免三个平台只在字数上不同，实际语言、结构和视觉也必须有明显差异。
- 使用同一个选题制作三平台对照结果，展示差异是否真实成立。
- 根据新采集样本持续更新偏好结论，避免平台画像长期不变。

## 交付内容

- 公众号、抖音和小红书的平台偏好画像。
- 各平台选题、标题、语言、结构和视觉偏好说明。
- 所有偏好结论对应的样本依据。
- 同一选题的三平台内容与画面对照结果。
- 平台偏好更新前后的生成效果对比。

## 完成标准

- 三个平台都有独立且可追溯的偏好画像。
- 同一选题在三个平台的标题、正文和画面有明显差异。
- 画面选择同时考虑选题内容和目标平台。
- 每条动态偏好都能够说明来自哪些样本。
- 偏好画像变化后，生成结果能够同步变化。
- 生成内容仍然符合校园和社团账号定位。
