# Prompt 迭代历史 - v3

## 创建日期
2026-07-07

## 版本说明
基于 v2 版本 3 轮 × 5 选题 = 15 条输出测试结果，针对 v2 暴露的核心遗留问题——**段落长度不稳定**进行专项优化。v2 测试数据：硬指标通过率 99%，肉眼均分 20.4/25，但干货类话题段落数波动大（6-24段），严重影响手机端阅读体验。

## v2 问题诊断与 v3 改进对照

| v2 问题 | v2 数据 | v3 改进措施 |
|---------|--------|------------|
| **段落长度不稳定（核心问题）** | 干货类话题段落数波动 6-24段，r3有3篇≤9段 | 新增「段落节奏」独立章节，明确要求 22-32 段、每段1-2句；给出正/反段落示范；要求每个要点拆成2-3个短段；自检清单新增段落数/句数检查 |
| few-shot 示例段落偏长 | v2 示例每段3-4句，模型学到长段 | 更新 few-shot 示例为严格1-2句/段的格式，标注段落计数说明 |
| 要点段落结构不明确 | 模型倾向把整个要点写在一段里 | 明确「观点段→展开段→操作/感受段」的三段式拆法 |
| cover_text 偶发偏差 | 1/15 不达标（8字） | 新增反面示例「招新实战避坑指南（8字）」；要求逐字清点字数；自检清单明确「逐字数过」 |
| 断句/标点偶发异常 | r2 test_02 出现句子粘连 | 自检清单新增「标点完整性」检查项 |
| risk_notes 未覆盖心理场景 | v2 r3 test_02 提到心理咨询中心是亮点 | 在 risk_notes 要求中新增「涉及心理情绪建议寻求咨询中心帮助」 |

## 核心设计变更

### 1. 段落节奏专项（新增独立模块，高优先级）

v2 虽然要求「每段不超过3句」，但没有给出段落数量目标和拆段示范，导致模型在干货类内容上倾向长段。

v3 方案：
- 明确段落数目标：**正文必须 22-32 段**
- 明确每段句数：**严格 1-2 句，最多不超过 3 句**
- 给出正反段落示范对比（正确的3段拆法 vs 错误的1段挤法）
- 要求每个要点必须拆成 2-3 个短段：观点段（1-2句）→展开段（1-2句）→操作/感受段（1-2句，可选）
- 正文要求章节新增「段落数不足时拆长段、过多时合并短句」的指引
- 禁忌中新增「禁止一个要点只写一个长段落」

### 2. few-shot 示例更新

v2 的 few-shot 示例段落略长（部分段 3 句），模型可能学到这个模式。

v3 方案：
- 更新示例为严格 1-2 句/段
- 示例后增加段落计数说明（「这段示例有12个段落，每段1-2句」）
- 明确标注「注意它的段落节奏」引导模型关注拆分方式

### 3. 自检清单升级（从 6 项扩展到 10 项）

v2 的 6 项自检：cover_text字数/body字数/emoji编号/第一人称/标题数字/JSON格式

v3 扩展为 10 项，新增：
- 第3项：段落数是否在 22-32 段
- 第4项：每段是否 1-2 句话
- 第5项：每个要点是否拆成 2-3 个短段
- 第9项：标点完整性检查（避免句子粘连）
- 将 cover_text 自检改为「逐字数过」

### 4. 其他微调

- cover_text 字数说明增加「不含竖线｜」（v2 中「招新爆款指南｜实操技巧全收录」含竖线算字数歧义）
- cover_text 反面示例新增「招新实战避坑指南（8字，差2字）」——这是 v2 r3 实际出现的错误
- 表达克制中区分「必须」在标题和正文中的不同处理（标题可适度使用）
- 情感连接要求「互动引导单独成段」
- risk_notes 场景新增「涉及心理情绪建议咨询中心」
- user_template.md 创作提醒从 6 条扩展到 7 条，增加段落数和要点拆分要求

## system.md 关键变更片段（v2→v3 对比）

### 段落节奏章节（v3 新增/强化）

v2 段落要求（分散在各处）：
```
6. 段落节奏：段落短，多用换行，适合手机竖屏阅读，每段不超过3句话，段落数建议20-35段
```

v3 段落要求（独立章节，带示范）：
```
### 6. 段落节奏（特别重要，严格遵守）
- 正文段落数必须在 22-32 段之间
- 每段严格 1-2 句话，最多不超过 3 句话
- 写完一个意思就换行，不要在一个段落里说多层意思
- 每个要点必须拆成 2-3 个小段落：观点段→展开段→操作/感受段
- ✅ 正确示范（一个要点拆成 3 个短段）：
  先说说预热这一步，别等摆摊当天才开始吆喝。

  提前三天在学校表白墙、年级群发预告，效果完全不同。
  我亲测过，提前预热的摊位，当天来咨询的人数直接翻倍。
- ❌ 错误示范（一个要点挤在一个长段）：
  先说说预热这一步，别等摆摊当天才开始吆喝，提前三天...
```

### 自检清单升级

v2（6项）：cover_text字数/body字数/emoji编号/第一人称/标题数字/JSON格式

v3（10项）：新增段落数/每段句数/要点拆分/标点完整性，cover_text改为逐字计数
