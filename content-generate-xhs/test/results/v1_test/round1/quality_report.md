## 五、v2 Prompt 迭代方向

基于本轮测试发现的问题，下一轮（v2）Prompt 迭代计划如下：

1. **封面文案约束强化**：明确给出 10-12 字的示例（如「社团招新转化率翻倍秘籍」），并要求模型自检字数。
2. **标题字数硬约束**：在输出 schema 中加入长度说明，或要求模型在 JSON 中附 title_lengths 字段自校验。
3. **绝对化词替换示范**：列出替代词表（最 → 很/相当/个人觉得最…之一；保证 → 亲测有效；第一 → 前排/优先）。
4. **风格锚定**：在 system.md 中增加 Few-shot 示例段落，固定「过来人第一人称聊天」风格，禁止使用 emoji 和工具号语气。
5. **风格一致性指令**：明确禁止使用「👇」「码住」「必看」等工具号高频词，统一到「分享/建议/聊聊」的口语体系。
